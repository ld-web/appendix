VSCode raccourcis clavier

Liste officielle

Palette de commande

Ctrl + Shift + P permet d’ouvrir la palette de commandes. Ensuite, taper le début d’un texte et VSCode proposera des commandes correspondantes.

Exemple

Ctrl + Shift + P puis reloa devrait proposer de recharger la fenêtre (Reload Window).

Remonter/Descendre une ligne

Dans l’éditeur, on peut remonter ou descendre une ligne complète avec :

Alt + ⬆️ ou Alt + ⬇️

Dupliquer un bloc de code

Pour dupliquer un bloc de code (ou une ligne seule), sélectionner le code à dupliquer, puis appuyer sur :

Alt + Shift + ⬇️

Supprimer une ligne complète

Quand le curseur se trouve sur une ligne, on peut supprimer la ligne :

Shift + Suppr

Commenter/Décommenter du code

Pour commenter du code, 2 possibilités :

  • Ctrl + :
  • Ctrl + K suivi de Ctrl + C

Pour décommenter du code, 2 possibilités :

  • Ctrl + :
  • Ctrl + K suivi de Ctrl + U

Chercher du texte dans le fichier courant

Pour chercher du texte dans le fichier courant :

Ctrl + F puis taper le texte recherché.

Chercher du texte dans tout le projet

Pour chercher du texte dans tout le projet et non dans le fichier courant :

Ctrl + Shift + F

Il existe différentes options, par exemple pour exclure des dossiers, etc…

Sélectionner le même texte

Pour sélectionner le même texte plusieurs fois au sein d’un fichier, on peut sélectionner le texte une première fois, puis utiliser Ctrl + D autant de fois que nécessaire pour sélectionner les différentes occurences de ce même texte au sein du fichier.

Si on veut sélectionner toutes les occurences du texte dans le fichier, on peut utiliser :

Ctrl + Shift + L

Ouvrir le terminal

Pour ouvrir le terminal :

Ctrl + ù

Pour ouvrir un nouveau terminal :

Ctrl + Shift + ù

Accéder à un fichier depuis un projet

Dans un projet, taper Ctrl + P puis le nom du fichier.

Quand le nom correct est suggéré par VSCode, taper Entrée pour l’ouvrir.
