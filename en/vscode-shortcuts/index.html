VSCode keyboard shortcuts

Official reference

Command palette

Ctrl + Shift + P opens the command palette. Then, start to type some text and VSCode will suggest corresponding commands.

Example

Ctrl + Shift + P then reloa should suggest to reload the current window (Reload Window).

Move line up/down

In the editor, one can move a whole line up or down with :

Alt + ⬆️ ou Alt + ⬇️

Duplicate code block

To duplicate a code block (or a single line), select the code and press :

Alt + Shift + ⬇️

Delete a line

When on a line, it’s possible to delete the line with :

Shift + Suppr

Comment/Uncomment code

To comment code, 2 ways :

  • Ctrl + :
  • Ctrl + K then Ctrl + C

To uncomment code, 2 ways :

  • Ctrl + :
  • Ctrl + K then Ctrl + U

Search text in the current file

To search text in the current file :

Ctrl + F then enter text to search.

Search text in the whole project

To search text in the project and not only in the current file :

Ctrl + Shift + F

There are some options, for instance to exclude some folders, etc…

Select same text

To select the same text multiple times in the same file, one can first select the text, and then use Ctrl + D to select other occurences of the same text into the file.

To select all occurences of the same text into the file, use :

Ctrl + Shift + L

Open terminal

To open terminal :

Ctrl + ù

To open a new terminal :

Ctrl + Shift + ù
